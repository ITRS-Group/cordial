# `gdna version`

The `gdna version` command returns details about the program version and, using the `--debug`/`-d` flag will return information about packages used.
## Usage

```text
gdna version
```

### Options

```text
  -f, --config FILE    Use configuration FILE
  -l, --logfile file   Write logs to file. Use '-' for console or /dev/null for none (default "docs.log")
```
