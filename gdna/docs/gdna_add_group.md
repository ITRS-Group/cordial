# `gdna add group`

Add an item to an include list

```text
gdna add group [FLAGS] CATEGORY NAME PATTERN...
```

### Options

```text
      --allocation int   add allocation (use -1 for no change) (default -1)
      --token string     token to allocate (default "server")
```
