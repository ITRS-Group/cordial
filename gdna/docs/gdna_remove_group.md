# `gdna remove group`

Remove an item from groups

## Usage

```text
gdna remove group [FLAGS] CATEGORY [NAME [PATTERN...]]
```

### Options

```text
  -A, --all            remove all filters or group for a category
  -f, --config FILE    Use configuration FILE
  -l, --logfile file   Write logs to file. Use '-' for console or /dev/null for none (default "docs.log")
```
