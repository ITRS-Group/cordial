# `geneos disable`

Disable instances

```text
geneos disable [TYPE] [NAME...] [flags]
```

Mark any matching instances as disabled. The instances are also stopped.

### Options

```text
  -S, --stop    Stop instances
  -F, --force   Force disable instances
```
