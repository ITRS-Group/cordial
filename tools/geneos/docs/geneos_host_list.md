# `geneos host list`

List hosts, optionally in CSV or JSON format

```text
geneos host list [flags] [NAME...]
```

List the matching remote hosts.

### Options

```text
  -a, --all      Show all hosts
  -j, --json     Output JSON
  -i, --pretty   Output indented JSON
  -c, --csv      Output CSV
```
