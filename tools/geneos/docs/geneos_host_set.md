# `geneos host set`

Set options on remote host configurations.

```text
geneos host set [flags] [NAME...] [KEY=VALUE...]
```

### Options

```text
  -p, --prompt               Prompt for password
  -P, --password PLAINTEXT   password
  -k, --keyfile KEYFILE      Keyfile
  -i, --privatekey PATH      Private key file
```
