# `ims-gateway start`



## Usage

```text
ims-gateway start [flags]
```

### Options

```text
  -D, --daemon         Daemonise the proxy process
  -l, --logfile file   Write logs to file. Use '-' for console or /dev/null for none (default "-")
  -c, --conf string    override config file
```
