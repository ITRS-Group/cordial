# `docs server`

Run server for config request

## Usage

```text
docs server [flags]
```

### Options

```text
      --config string    path to configuration file
  -D, --daemon           Run as a daemon
  -L, --logfile string   Override configured log file path
  -N, --nowatch          Do not watch configuration file for changes
```
